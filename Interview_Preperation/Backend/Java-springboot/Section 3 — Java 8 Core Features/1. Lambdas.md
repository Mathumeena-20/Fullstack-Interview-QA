# ‚úÖ 1. What Is a Lambda Expression in Java 8?

A **lambda expression** is a short block of code (a function) that can be passed around as an object.

‚úî Introduced in Java 8
‚úî Used to implement **functional interfaces** (interfaces with exactly one abstract method)

### Example:

```java
Runnable r = () -> System.out.println("Hello Lambda");
r.run();
```

Equivalent to:

```java
Runnable r = new Runnable() {
    public void run() {
        System.out.println("Hello Lambda");
    }
};
```

# ‚úÖ 2. Why Were Lambdas Introduced in Java?

### Java introduced lambdas to:

‚úî Reduce boilerplate code
‚úî Enable functional programming
‚úî Improve readability
‚úî Enhance parallel operations using Streams
‚úî Treat behavior as data (pass functions as arguments)

Example with Streams:

```java
List<Integer> list = Arrays.asList(1,2,3,4);
list.forEach(n -> System.out.println(n));
```

# ‚úÖ 3. Lambda vs Anonymous Inner Class

| Feature                  | Lambda Expression       | Anonymous Inner Class                |
| ------------------------ | ----------------------- | ------------------------------------ |
| Syntax                   | Short, concise          | Longer                               |
| `this` keyword           | Refers to *outer* class | Refers to *current anonymous object* |
| Performance              | Faster (invokedynamic)  | Slower                               |
| Extra classes generated? | ‚ùå No                    | ‚úî Yes (.class file)                  |
| Use case                 | Functional interfaces   | Interfaces with multiple methods     |

### Example difference:

```java
class Test {
    public void show() {
        Runnable r1 = new Runnable() {
            public void run() {
                System.out.println(this); // refers to anonymous object
            }
        };

        Runnable r2 = () -> System.out.println(this); // refers to Test class
    }
}
```

# ‚úÖ 4. Syntax of a Lambda Expression

### Basic Syntax:

```
(parameters) -> expression
(parameters) -> { statements }
```

### Examples:

```java
() -> System.out.println("No params");

x -> x * x;

(a, b) -> a + b;

(name) -> { 
    System.out.println("Hello " + name);
};
```

# ‚úÖ 5. Can Lambdas Access Local Variables? What Are the Restrictions?

Yes, lambdas can access **final** or **effectively final** variables.

### Example:

```java
public class Test {
    public static void main(String[] args) {
        int num = 10;  // effectively final

        Runnable r = () -> System.out.println(num); 
        r.run();
    }
}
```

### ‚ùå Cannot modify `num` afterward:

```java
num = 20;  // ERROR: variable must be effectively final
```

### Why restriction?

Because lambda may run **later** and local variables live on the **stack**, which may not exist then.

# ‚úÖ 6. What Is Variable Capture?

When a lambda uses variables from its enclosing scope.

Example:

```java
int factor = 2;

Function<Integer, Integer> multiply = x -> x * factor;

System.out.println(multiply.apply(5)); // 10
```

Here, `factor` is "captured" by the lambda.

# ‚úÖ 7. Are Lambda Expressions Objects in Java?

YES, but not like traditional objects.

‚úî Created using **invokedynamic**
‚úî No anonymous classes generated
‚úî More lightweight

Example:

```java
Predicate<Integer> p = x -> x > 0;
```

This is internally an object implementing `Predicate`.

---

# ‚úÖ 8. How Are Lambdas Implemented Internally by JVM?

They use:

### ‚úî `invokedynamic` bytecode (added in Java 7)

### ‚úî Dynamic lambda creation at runtime

### ‚úî No new class files generated

This makes lambdas:

* Faster
* Memory-efficient
* More optimized than anonymous classes

# ‚úÖ 9. Can We Throw Checked Exceptions Inside Lambdas?

Yes, **only if** the functional interface declares it.

### Example ‚Äì Allowed:

```java
@FunctionalInterface
interface MyInterface {
    void test() throws Exception;
}

MyInterface obj = () -> {
    throw new Exception("Checked exception");
};
```

### Example ‚Äì Not allowed:

```java
Runnable r = () -> {
    throw new Exception("Error");   // ERROR: Runnable.run() doesn't throw checked exception
};
```

# ‚úÖ 10. Real Project Use-Case of Lambdas

### Example: Filtering active users using Streams

```java
List<User> users = getUsersFromDB();

List<User> activeUsers =
        users.stream()
             .filter(u -> u.isActive())
             .collect(Collectors.toList());
```

### Example: Sorting

```java
Collections.sort(users,
    (u1, u2) -> u1.getName().compareTo(u2.getName()));
```

### Example: REST API async calls (CompletableFuture)

```java
CompletableFuture.supplyAsync(() -> service.getData())
                 .thenAccept(data -> System.out.println(data));
```

### Example: Runnable in thread

```java
new Thread(() -> System.out.println("Running thread...")).start();
```
