# 1. Difference between `LocalDate`, `LocalTime`, and `LocalDateTime`

| Class           | Represents             | Example               |
| --------------- | ---------------------- | --------------------- |
| `LocalDate`     | Date only (YYYY-MM-DD) | `2025-02-15`          |
| `LocalTime`     | Time only (HH:MM:SS)   | `10:15:30`            |
| `LocalDateTime` | Date + Time            | `2025-02-15T10:15:30` |

### Examples:

```java
LocalDate date = LocalDate.now();
LocalTime time = LocalTime.now();
LocalDateTime dt = LocalDateTime.now();
```

# 2. Difference: Old Date API vs Java 8 Date-Time API

| Old API (`java.util.Date`, `Calendar`) | Java 8 API (`java.time.*`) |
| -------------------------------------- | -------------------------- |
| Mutable ❌                              | Immutable ✔                |
| Not thread-safe ❌                      | Thread-safe ✔              |
| Confusing months (0-based) ❌           | Clear (1–12) ✔             |
| Poor formatting/parsing ❌              | `DateTimeFormatter` ✔      |
| Poor timezone handling ❌               | Strong timezone support ✔  |
| Hard to understand ❌                   | Clean, modern API ✔        |

### Example (old):

```java
Date d = new Date();   // Mutable, unsafe
```

### Example (Java 8+):

```java
LocalDate d = LocalDate.now();  // Immutable, safe
```

# 3. How do you format `LocalDateTime`?

Using `DateTimeFormatter`.

### Example:

```java
LocalDateTime dt = LocalDateTime.now();

String formatted = dt.format(DateTimeFormatter.ofPattern("dd-MM-yyyy HH:mm"));
System.out.println(formatted);
```

Output:

```
15-02-2025 14:45
```

# 4. What is `Period` and `Duration`?

### `Period`

Represents **date-based** values (years, months, days).

Used with **LocalDate**.

```java
Period p = Period.between(LocalDate.of(2023,1,1), LocalDate.now());
System.out.println(p.getYears() + " years " + p.getMonths() + " months");
```

---

### `Duration`

Represents **time-based** values (hours, minutes, seconds).

Used with **LocalTime / LocalDateTime**.

```java
Duration d = Duration.between(LocalTime.of(10, 0), LocalTime.of(12, 30));
System.out.println(d.toMinutes());  // 150
```

---

# 5. How to calculate difference between two `LocalDate` values?

### Example:

```java
LocalDate start = LocalDate.of(2023, 1, 1);
LocalDate end = LocalDate.of(2025, 2, 15);

Period diff = Period.between(start, end);

System.out.println(diff.getYears());   // 2
System.out.println(diff.getMonths());  // 1
System.out.println(diff.getDays());    // 14
```

# 6. How to get current date and time?

```java
LocalDate date = LocalDate.now();
LocalTime time = LocalTime.now();
LocalDateTime dateTime = LocalDateTime.now();
Instant instant = Instant.now();  // machine timestamp
```

---

# 7. What is `ZonedDateTime`?

Represents a date + time **with a timezone**.

Used for global applications (e.g., flight bookings, banking).

### Example:

```java
ZonedDateTime zdt = ZonedDateTime.now(ZoneId.of("America/New_York"));
System.out.println(zdt);
```

# 8. Difference between `Instant` and `LocalDateTime`

| Instant                   | LocalDateTime               |
| ------------------------- | --------------------------- |
| Timestamp from Unix epoch | Human-readable date & time  |
| Always UTC                | No timezone                 |
| Good for logs, timestamps | Good for UI, business logic |
| Machine time              | Human time                  |

### Example:

```java
Instant i = Instant.now();          // UTC
LocalDateTime ldt = LocalDateTime.now(); // System timezone
```

# 9. Thread-safety of Java 8 Date-Time API

✔ **Fully thread-safe**
✔ All classes are **immutable**
✔ Avoids concurrency issues that occurred with `Date` and `Calendar`

Example:

```java
// Safe to use across threads
LocalDate date = LocalDate.now();
```

# 10. Real-World Use Case of Java 8 Date-Time API

### ✔ Banking / Finance

Convert timestamps across timezones:

```java
ZonedDateTime india = ZonedDateTime.now(ZoneId.of("Asia/Kolkata"));
ZonedDateTime usa = india.withZoneSameInstant(ZoneId.of("America/Chicago"));
```

### ✔ Scheduler / Cron jobs

Calculate next run:

```java
LocalDateTime nextRun = LocalDateTime.now().plusHours(2);
```

### ✔ Booking systems

Find duration between flights:

```java
Duration duration = Duration.between(departure, arrival);
```

### ✔ Age calculation

```java
Period age = Period.between(birthDate, LocalDate.now());
```
