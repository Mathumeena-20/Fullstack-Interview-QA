## 1. What is a Lambda Expression in Java 8?

A **lambda expression** is an anonymous function that provides a concise way to implement functional interfaces.

### Example:

```java
Runnable r = () -> System.out.println("Hello Lambda");
r.run();
```

---

## 2. Why were Lambdas introduced in Java?

Because Java needed:

‚úÖ Functional programming support
‚úÖ Less boilerplate code
‚úÖ Easy parallel programming with Streams
‚úÖ Cleaner and readable code for callbacks

---

## 3. Lambda vs Anonymous Inner Class

| Feature        | Lambda Expression  | Anonymous Inner Class |
| -------------- | ------------------ | --------------------- |
| Code length    | Short              | Long                  |
| `this` keyword | Refers outer class | Refers inner class    |
| Boilerplate    | Very less          | More                  |
| Performance    | Better             | Slightly slower       |

### Example:

```java
// Anonymous class
Runnable r1 = new Runnable() {
    public void run() {
        System.out.println("Hello");
    }
};

// Lambda
Runnable r2 = () -> System.out.println("Hello");
```

---

## 4. Syntax of Lambda Expression

### General syntax:

```java
(parameters) -> { body }
```

### Examples:

```java
// No parameter
() -> System.out.println("Hello");

// One parameter
x -> x * x;

// Multiple parameters
(a, b) -> a + b;
```

---

## 5. Can Lambda Access Local Variables?

‚úÖ Yes, but only **final or effectively final** variables.

Example:

```java
int x = 10;

Runnable r = () -> {
    System.out.println(x);  
};

// x = 20; ‚ùå Not allowed (not effectively final)
```

---

## 6. What is Variable Capture?

When a lambda uses variables from outside its body.

Example:

```java
int number = 10;

Consumer<Integer> c = (n) -> System.out.println(n + number);
```

Here, `number` is **captured variable**.

---

## 7. Are Lambda Expressions Objects?

‚úÖ Technically yes. They are objects but:

* They don‚Äôt create separate `.class` files.
* They are implemented using `invokedynamic`.

---

## 8. How are Lambdas Implemented Internally?

JVM uses:
‚úÖ `invokedynamic` bytecode instruction
‚úÖ Functional interface mapping
‚úÖ Runs faster than anonymous classes

Lambdas don‚Äôt generate extra class files; they are generated dynamically.

---

## 9. Can we throw Checked Exceptions inside Lambdas?

Yes, but only if the functional interface method declares it.

Example:

```java
@FunctionalInterface
interface MyInterface {
    void read() throws Exception;
}

MyInterface obj = () -> {
    throw new Exception("Checked Exception");
};
```

If the interface doesn‚Äôt declare, compiler error occurs.

## 10. Real Project Use-case of Lambdas

### Scenario: Filter active users

```java

List<String> activeUsers = users.stream()
                                .filter(status -> status.equals("Active"))
                                .toList();

System.out.println(activeUsers);
```
