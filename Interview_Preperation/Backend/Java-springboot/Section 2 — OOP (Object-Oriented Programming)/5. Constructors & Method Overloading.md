## 1. What is a Constructor?

A **constructor** is a special block of code used to **initialize an object** when it is created.

### Key points:

* Name must be same as class name
* No return type (not even `void`)
* Called automatically during object creation

### Example:

```java
class Student {
    String name;

    Student(String name) {   // Constructor
        this.name = name;
    }

    void display() {
        System.out.println(name);
    }
}

public class Test {
    public static void main(String[] args) {
        Student s = new Student("Rahul");
        s.display();   // Rahul
    }
}
```

---

## 2. Difference between Constructor and Method

| Constructor                   | Method                     |
| ----------------------------- | -------------------------- |
| Used to initialize object     | Used to perform operations |
| Name same as class            | Can have any name          |
| No return type                | Has a return type / void   |
| Automatically called          | Must be called explicitly  |
| Executes only once per object | Can execute many times     |

## 3. Types of Constructors

### 1. Default Constructor

Provided by Java if no constructor is defined.

```java
class Car {
    int speed;
}

// Java creates default constructor internally
```

---

### 2. No-Argument Constructor

Created explicitly by programmer.

```java
class Car {
    Car() {
        System.out.println("No-arg constructor");
    }
}
```

---

### 3. Parameterized Constructor

Accepts values at object creation.

```java
class Car {
    int speed;

    Car(int speed) {
        this.speed = speed;
    }
}
```

---

## 4. What happens if you don’t define a constructor?

Java compiler automatically provides a **default constructor**, which:

* Assigns default values
* Calls parent class constructor

But if you write **any constructor**, Java will **not generate** a default one.

---

## 5. Can constructors be overloaded?

✅ **Yes.**
Multiple constructors with different parameters.

### Example:

```java
class Student {
    String name;
    int age;

    Student() {
        name = "Unknown";
    }

    Student(String name) {
        this.name = name;
    }

    Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

---

## 6. Can constructors be overridden?

❌ **No.**
Because:

* Constructors are not inherited
* Overriding only applies to methods

---

## 7. What is Constructor Chaining?

Calling one constructor from another using:

* `this()` → same class constructor
* `super()` → parent class constructor

### Example:

```java
class A {
    A() {
        System.out.println("Parent Constructor");
    }
}

class B extends A {
    B() {
        super();  // Calls parent constructor
        System.out.println("Child Constructor");
    }
}
```

Output:

```
Parent Constructor
Child Constructor
```

---

## 8. What is `this()` and `super()`?

| Keyword   | Use                                     |
| --------- | --------------------------------------- |
| `this()`  | Calls another constructor in same class |
| `super()` | Calls constructor of parent class       |

⚠ Must be the **first statement** inside constructor.

---

## 9. Rules of Method Overloading

Method overloading means:
Multiple methods with same name but different parameters.

### Rules:

1. Same method name
2. Different parameter list (type, number, or order)
3. Return type change alone is not enough

### ✅ Valid Overloading:

```java
class Calculator {
    int add(int a, int b) {
        return a + b;
    }

    double add(double a, double b) {
        return a + b;
    }

    int add(int a, int b, int c) {
        return a + b + c;
    }
}
```

### ❌ Invalid:

```java
int add(int a, int b);
double add(int a, int b); // Error, only return type differs
```

## 10. Can we overload `main()` method?

✅ Yes, **but JVM only executes this one**:

```java
public static void main(String[] args)
```

### Example:

```java
class Test {

    public static void main(String[] args) {
        System.out.println("Main with String[]");
        main(10);
    }

    public static void main(int a) {
        System.out.println("Overloaded main: " + a);
    }
}
```

### Output:

```
Main with String[]
Overloaded main: 10
```
