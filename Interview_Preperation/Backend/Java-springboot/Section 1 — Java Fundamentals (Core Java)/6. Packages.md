## 1. What is a Package in Java?

A **package** is a namespace that organizes related classes and interfaces into a folder-like structure.

### Example:

```java
package com.company.project;

public class Employee {
    public void show() {
        System.out.println("Employee class");
    }
}
```

---

## 2. Difference between Package and Module (Java 9+)

| Package                      | Module                                |
| ---------------------------- | ------------------------------------- |
| Groups related classes       | Groups related packages               |
| Exists since early Java      | Introduced in Java 9                  |
| Organizes code for structure | Controls dependency and accessibility |
| No strong encapsulation      | Enables strong encapsulation          |

### Example:

```java
// module-info.java
module com.mycompany.project {
    exports com.company.project;
}
```

---

## 3. Why are Packages Used?

✅ Avoid class name conflicts
✅ Organize large applications
✅ Control access using access modifiers
✅ Helps in reuse and maintenance

---

## 4. Difference between public, default and protected access modifiers

| Modifier  | Same Class | Same Package | Subclass | Other Package                |
| --------- | ---------- | ------------ | -------- | ---------------------------- |
| public    | ✅          | ✅            | ✅        | ✅                            |
| default   | ✅          | ✅            | ❌        | ❌                            |
| protected | ✅          | ✅            | ✅        | ✅ (only through inheritance) |

---

## 5. What is Classpath?

Classpath tells JVM where to find `.class` files and external libraries.

Example:

```bash
java -cp myfolder;lib.jar MainClass
```

---

## 6. Can a package have sub-packages?

✅ Yes.

Example:

```
com.company
com.company.project
com.company.project.service
```

---

## 7. Difference between import and static import

### import

Used to import classes.

```java
import java.util.Scanner;
```

### static import

Imports static members of a class.

```java
import static java.lang.Math.PI;

System.out.println(PI);
```

---

## 8. How does JVM locate a class using package name?

Example:

```java
package com.example.test;
```

Directory structure must be:

```
com/example/test/ClassName.class
```

JVM uses:

1. Classpath
2. Package name
3. Folder structure

---

## 9. Can a class belong to multiple packages?

❌ No.
A class can belong to only one package.

---

## 10. How to create a custom package?

### Step 1 – Define package:

```java
package mypackage;

public class Demo {
    public void show() {
        System.out.println("Hello");
    }
}
```

### Step 2 – Compile:

```bash
javac -d . Demo.java
```

It creates folder:

```
mypackage/Demo.class
```

### Step 3 – Use it:

```java
import mypackage.Demo;

public class Main {
    public static void main(String[] args) {
        Demo d = new Demo();
        d.show();
    }
}
```
