# ‚úÖ **1. Difference between attribute selectors and pseudo-classes**

### **Attribute selectors**

Select elements **based on attributes or attribute values**.

Examples:

```css
input[type="text"] { ... }
a[target="_blank"] { ... }
img[alt] { ... }
```

They match **static HTML attributes**.

---

### **Pseudo-classes**

Select elements based on **state or condition** (dynamic behavior).

Examples:

```css
a:hover { ... }       /* on hover */
input:focus { ... }   /* when focused */
li:first-child { ... }
```

They depend on **user interaction or document structure**.

---

### Summary

| Selector Type       | Matches                                |
| ------------------- | -------------------------------------- |
| Attribute selectors | Attribute presence or value            |
| Pseudo-classes      | Dynamic state or structural conditions |

---

# ‚úÖ **2. Difference between `:nth-child()` and `:nth-of-type()`**

---

### **`:nth-child(n)`**

Selects the **n-th child among ALL children**, regardless of tag type.

Example:

```html
<div>
  <p></p>
  <span></span>
  <p></p>
</div>
```

```css
p:nth-child(1) { color: red; }  /* ‚ùå Not matched (1st child is <p> but counts all types) */
p:nth-child(3) { color: red; }  /* ‚úÖ Matches 3rd child */
```

---

### **`:nth-of-type(n)`**

Selects the **n-th child of the SAME type only**.

```css
p:nth-of-type(1) { color: blue; } /* First <p> inside parent */
p:nth-of-type(2) { color: red; }  /* Second <p> */
```

---

### Summary

| Selector         | Based On                     |
| ---------------- | ---------------------------- |
| `:nth-child()`   | Position among all siblings  |
| `:nth-of-type()` | Position among same tag type |

# ‚úÖ **3. What are pseudo-elements (`::before`, `::after`)?**

Pseudo-elements **create virtual elements** that do not exist in HTML.
Used for icons, decorations, extra content.

### Example

```css
.button::before {
  content: "üî•";
  margin-right: 5px;
}

.card::after {
  content: "";
  display: block;
  height: 2px;
  background: #000;
}
```

### Uses:

* Custom bullets
* Background overlays
* Decorative icons
* Underlines / shapes

Pseudo-elements **require `content:`**.

# ‚úÖ **4. Difference between `*` selector and `:root` selector**

### **`*` ‚Äî Universal Selector**

Selects **all elements**.

```css
* {
  margin: 0;
  padding: 0;
}
```

Used for resets.

---

### **`:root` ‚Äî Highest root element**

In HTML: `:root` = `<html>` element.

```css
:root {
  --primary-color: #333;
  font-size: 16px;
}
```

Used for:

* CSS variables
* Global fonts
* Global theme settings

---

### Summary

| Selector | Meaning                        |
| -------- | ------------------------------ |
| `*`      | Selects every element          |
| `:root`  | Selects the document root only |

# ‚úÖ **5. What is the `>` vs `+` vs `~` selectors?**

---

### **`>` ‚Äî Child Selector**

Matches **direct children only**.

```css
div > p { color: red; }
```

Matches:

```html
<div><p></p></div>         <!-- direct child -->
```

Does NOT match:

```html
<div><span><p></p></span></div>   <!-- nested deeper -->
```

---

### **`+` ‚Äî Adjacent Sibling Selector**

Selects the **next immediate sibling** only.

```css
h1 + p { color: blue; }
```

Matches only the **first p after h1**.

---

### **`~` ‚Äî General Sibling Selector**

Matches **all siblings after the element**, not just the first.

```css
h1 ~ p { color: green; }
```

Matches **every p** that comes after h1 (same parent).

---

### Summary table

| Selector | Selects                         |
| -------- | ------------------------------- |
| `>`      | Direct child                    |
| `+`      | First sibling immediately after |
| `~`      | All siblings after              |

# ‚úÖ **6. What is the specificity order?**

CSS resolves conflicts using specificity.

### Specificity weight:

| Selector Type                         | Points   |
| ------------------------------------- | -------- |
| Inline styles                         | **1000** |
| ID selectors                          | **100**  |
| Classes / attributes / pseudo-classes | **10**   |
| Elements / pseudo-elements            | **1**    |
| Universal selector (*)                | **0**    |

---

### Example comparison:

```css
#title {}              /* 100 */
.title {}              /* 10 */
h1 {}                  /* 1 */
h1.title {}            /* 11 */
a:hover {}             /* 10 */
```

### Final Order (highest ‚Üí lowest specificity)

1. **Inline styles**
2. **ID selectors**
3. **Class, attribute, pseudo-class selectors**
4. **Element selectors**
5. **Universal selector `*`**
