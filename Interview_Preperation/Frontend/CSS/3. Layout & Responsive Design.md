# âœ… **1. What is Mobile-First vs Desktop-First Design?**

### **Mobile-First**

* Start designing for **small screens first**
* Then add styles for larger screens
* Uses **min-width** media queries

```css
/* Base = mobile */
.box {
  font-size: 14px;
}

/* Larger screens */
@media (min-width: 768px) {
  .box { font-size: 18px; }
}
```

### **Desktop-First**

* Start with **desktop styles**
* Then adjust for smaller screens
* Uses **max-width** media queries

```css
/* Base = desktop */
.box {
  font-size: 20px;
}

/* Tablet / mobile */
@media (max-width: 768px) {
  .box { font-size: 16px; }
}
```

### **Rule**

ðŸ“Œ **Mobile-first is preferred today** (better performance, simpler logic).

# âœ… **2. How do media queries work? + Example**

### Example

# âœ… **4. What is `calc()` used for?**

Allows performing calculations inside CSS.

Used for:

* Responsive layouts
* Dynamic sizes
* Combining units (`px`, `%`, `vh`)

### Example

```css
.box {
  width: calc(100% - 50px);
  padding: calc(2rem + 10px);
}
```

# âœ… **5. How does `object-fit` work?**

Controls **how images or videos fit inside containers**.

### **Common values**

| Value     | Meaning                               |
| --------- | ------------------------------------- |
| `cover`   | Fills container, cropping allowed     |
| `contain` | Fits entire image, keeps aspect ratio |
| `fill`    | Stretches, may distort                |
| `none`    | No scaling                            |

### Example

```css
img {
  width: 300px;
  height: 200px;
  object-fit: cover;
}
```

# âœ… **6. What is `aspect-ratio` in CSS?**

Maintains a consistent **width-to-height ratio**, regardless of content.

### Example

```css
.box {
  aspect-ratio: 16 / 9;
  width: 400px;
  background: lightgray;
}
```

| Width | Height                |
| ----- | --------------------- |
| 400px | automatically = 225px |

Very useful for:

* Videos
* Cards
* Image placeholders
